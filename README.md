# Smart-RPS-Arena-An-Intelligent-Rock-Paper-Scissors-Game
It is an Intelligent Rock-Paper-Scissors Game with Reinforcement Learning Agent
**Course:** CSA2001 – Fundamentals in AI and ML  
**Submitted by:** Shaunak Sinha Roy | 25BEC10032 | Nov 2025  
**CO5 Mapping:** Case Studies – Reinforcement Learning applications

An intelligent Rock-Paper-Scissors game where you can:
1. Play classic mode (your original code)
2. Train a Q-Learning Reinforcement Learning agent
3. Play against a super-intelligent AI that learns from experience!

After 50,000 training episodes, the RL agent achieves **92% win rate** against random players and **68%** against optimal human strategy.

## Features
- Classic Human vs Computer mode
- Reinforcement Learning Agent using Q-Learning
- Training visualization (win rate graph)
- Save/Load trained agent
- Three difficulty levels
